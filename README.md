# hybx_led_utilities

HybX LED Utilities for the Espressif IDF — reusable LED control functions for
standard GPIO LEDs and RGB LEDs via LEDC PWM, developed under the Hybrid RobotiX
(HybX) ecosystem.

See [docs/DESIGN.md](docs/DESIGN.md) for full design documentation and API reference.
